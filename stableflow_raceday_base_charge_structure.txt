
# StableFlow — Raceday Base Charges Structure

## New Feature Added Concept

Raceday charges should support:

- global default charges
- racecourse-specific default charges
- automatic pre-fill when a racecourse is selected

---

# Recommended Structure

## New Table:
`racecourse_base_charges`

Fields:

- ID
- Racecourse
- Charge Name
- Category
- Default Amount
- Billable
- Notes
- Active

---
